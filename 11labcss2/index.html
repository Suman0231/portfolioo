CSS Tutorial – Ways to Add CSS

1. Types of CSS Adding Methods

There are three ways of inserting a style sheet:

  • Inline CSS – use the style attribute inside HTML elements
  • Internal CSS – use the <style> element inside the head section
  • External CSS – link an external .css file

2. Inline CSS

An inline style is used to apply a unique style to a single element.

<html>
<body>

<h1 style="color:blue;text-align:center;">This is a heading</h1>
<p style="color:red;">This is a paragraph.</p>

</body>
</html>
Output:

This is a heading

This is a paragraph.

3. Internal CSS

An internal style sheet is used when a single HTML page has a unique style.

<html>
<head>
<style>
body {
  background-color: linen;
}

h1 {
  color: maroon;
  margin-left: 40px;
}
</style>
</head>

<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>
Output:

This is a heading

This is a paragraph.

4. External CSS

An external style sheet allows you to change the entire website by editing one file.

<html>
<head>
<link rel="stylesheet" href="mystyle.css">
</head>

<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

External CSS file (mystyle.css):

body {
  background-color: lightblue;
}

h1 {
  color: navy;
  margin-left: 20px;
}
Output:

This is a heading

This is a paragraph.

5. Summary

  • Inline CSS → best for small changes
  • Internal CSS → for single page styling
  • External CSS → best for large websites

6. YOUR TASK

🧩 Part 1: Starter HTML

Use the following base code and write your CSS inside the <style> section:

<!DOCTYPE html>
<html>
<head>
    <title>My First CSS Page</title>

    <style>
        /* Write your CSS here */
    </style>

</head>
<body>

    <h1 id="main-title">Welcome to My Page</h1>

    <p class="intro">This is my first paragraph.</p>
    <p>This is another paragraph.</p>

    <h2 class="section">About Me</h2>
    <p class="intro">I am learning CSS!</p>

    <button id="btn">Click Me</button>

</body>
</html>

    Change all <p> text color to blue
    Set font size to 18px

  2. Style using ID: #main-title
    Change the title color
    Center align it

  3. Style using class
    Make .intro text italic
    Change its color

  4. Add alignment
    Center align all headings (h1, h2)

  5. Style the button
    Change background color
    Change text color
    Add padding
