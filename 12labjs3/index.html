3. Methods of Adding JavaScript Code to HTML

1. Embedding JavaScript (Internal JS)

JavaScript code is written inside the <script> tag within the same HTML file. 

<!DOCTYPE html>
<html>
<head>
    <title>Embedded JavaScript</title>
</head>
<body>

<h2>Embedded JavaScript Example</h2>

<script>
    function showMessage() {
        document.write("Hello from Embedded JavaScript");
    }
    showMessage();
</script>

</body>
</html>

2. External JavaScript

JavaScript code is written in a separate .js file and linked to the HTML file. This is the best practice. 

<!-- HTML File -->
<!DOCTYPE html>
<html>
<head>
    <title>External JavaScript</title>
    <script src="script.js"></script>
</head>
<body>

<h2>External JavaScript Example</h2>

</body>
</html>

/* script.js */
function displayMessage() {
    document.write("Hello from External JavaScript");
}
displayMessage();

3. Inline JavaScript

JavaScript code is written directly inside an HTML element using events like onclick. 

<!DOCTYPE html>
<html>
<head>
    <title>Inline JavaScript</title>
</head>
<body>

<h2>Inline JavaScript Example</h2>

<button onclick="alert('Hello from Inline JavaScript')">
    Click Me
</button>

</body>
</html>
