JavaScript – Document Object Model (DOM)

1. Introduction to Document Object Model

The Document Object Model (DOM) represents the complete HTML document. When an HTML page is loaded in the browser, it becomes a document object. It is the root element of the web page and allows JavaScript to dynamically access and modify HTML content, attributes, and styles.

  • JavaScript can change HTML elements
  • JavaScript can change HTML attributes
  • JavaScript can change CSS styles
  • JavaScript can add or remove HTML elements
  • JavaScript can react to HTML events

2. Methods of Document Object

The document object provides several methods that allow JavaScript to access and manipulate HTML elements dynamically.

Some important methods of the document object are listed below.

  • write() – writes content to the document
  • writeln() – writes content with a newline
  • getElementById() - returns the element having the given id value.
  • getElementsByName() - returns the element having the given name value.
  • getElementsByTagName() - returns the element having the given TagName.
  • getElementsByClassName() - returns the element having the given id ClassName.

3. Accessing Field Value using Document Object

In this example, we retrieve the value entered by the user in a text field.

We use document.form1.name.value to access the input value where:

document – represents the HTML document
form1 – name of the form
name – name attribute of the input field
value – property that returns the entered value

This example displays a welcome message using the entered name. 

<script type="text/javascript">
function printvalue()
{
    var name = document.form1.name.value;
    alert("Welcome: " + name);
}
</script>

<form name="form1">
Enter Name:
<input type="text" name="name">
<input type="button" value="Print Name" onclick="printvalue()">
</form>

4. document.getElementById() Method

The document.getElementById() method returns the element with a specified ID.

This method is commonly used to access input fields, paragraphs, and other HTML elements by their unique ID.

The returned element can be modified using properties like innerHTML. 

<html>
<body>

<p id="demo"></p>
<p id="try"></p>

<script>
var x = "How are you";
document.getElementById("demo").innerHTML = x;
document.getElementById("try").innerHTML = "I am fine and you";
</script>

</body>
</html>

5. Sum of Two Numbers using getElementById()

This example demonstrates how to calculate the sum of two numbers entered by the user.

The values are retrieved using getElementById(), converted into numbers, and then added together.

The result is displayed in another input field. 

<html>
<head>
<title>>Addition of Two Numbers</title>

<script>
function add()
{
    var a = Number(document.getElementById("first").value);
    var b = Number(document.getElementById("second").value);
    var c = a + b;
    document.getElementById("answer").value = c;
}
</script>
</head>

<body>
Enter First Number:
<input type="number" id="first"><br>

Enter Second Number:
<input type="number" id="second"><br>

<button onclick="add()">Add</button><br><br>

Result:
<input type="number" id="answer">
</body>
</html>

6. document.getElementsByTagName() Method

The document.getElementsByTagName() method returns a collection of all elements with the specified tag name.

In this example, we count the total number of paragraph tags present in the document using this method. 

<script>
function countpara()
{
    var totalpara = document.getElementsByTagName("p");
    alert("Total paragraphs are: " + totalpara.length);
}
</script>

<p>This is first paragraph</p>
<p>This is second paragraph</p>
<p>This is third paragraph</p>

<button onclick="countpara()">Count Paragraph</button>

8. JavaScript Events

An event is a change in the state of an object.

In HTML, events represent actions performed by the user or the browser, such as clicking a button, pressing a key, or submitting a form.

When JavaScript code is included in HTML, it can react to these events and execute specific code. This process is known as Event Handling.

JavaScript handles HTML events using event handlers.

9. Mouse Events

Mouse events occur when the user interacts with a webpage using the mouse, such as clicking, moving, or hovering over an element. 

<html>

function mouseout()

    document.getElementById("msg").innerHTML = 
    "Mouse Out Event Triggered";
}

</body>
</html>

10. Keyboard Events

Keyboard events occur when the user presses or releases a key on the keyboard. Common keyboard events are onkeydown and onkeyup. 

<html>
<body>

Enter Text:
<input type="text" onkeydown="keyDown()" onkeyup="keyUp()">

<p id="keymsg"></p>

<script>
function keyDown()
{
    document.getElementById("keymsg").innerHTML =
    "Key Pressed (Key Down Event)";
}

function keyUp()
{
    document.getElementById("keymsg").innerHTML =
    "Key Released (Key Up Event)";
}
</script>

</body>
</html>

11. Form Events

Form events are triggered when the user interacts with form elements such as input fields, dropdowns, or submit buttons. 

<html>
<body>

<form onsubmit="return submitForm()">

Name:
<input type="text" 
       onfocus="focusEvent()" 
       onblur="blurEvent()">

<br><br>
<input type="submit" value="Submit">

</form>

<p id="formmsg"></p>

<script>
function focusEvent()
{
    document.getElementById("formmsg").innerHTML =
    "Input field is focused";

    "Input field lost focus";
}

{
    document.getElementById("formmsg").innerHTML =
    "Form submitted successfully";
    return false;

</script>
