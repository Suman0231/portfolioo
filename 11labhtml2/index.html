HTML Tables & Forms Tutorial

1. Basic HTML Table

This is the basic structure of an HTML table:

CODE:
<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>20</td>
  </tr>
</table>
Output:
Name Age
John 20

2. Table Tags Explained

CODE:
<table>  → Creates a table
<tr>     → Table row
<th>     → Table header cell
<td>     → Table data cell

3. Table with Style

CODE:
<table style="width:100%; border-collapse:collapse;" border="1">
  <tr>
    <th>Product</th>
    <th>Price</th>
  </tr>
  <tr>
    <td>Book</td>
    <td>$10</td>
  </tr>
</table>
Output:
Product Price
Book $10

4. Colspan & Rowspan

CODE:
<table border="1">
  <tr>
    <th colspan="2">Student Info</th>
  </tr>
  <tr>
    <td>Name</td>
    <td>Age</td>
  </tr>
</table>
Output:
Student Info
Name Age

5. Basic HTML Form

Forms are used to collect user input.

CODE:
<form>
  Name: <input type="text"><br><br>
  Email: <input type="email"><br><br>
  <input type="submit" value="Submit">
</form>
Output:
Name:

Email:
