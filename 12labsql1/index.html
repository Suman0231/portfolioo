SQL Tutorial – Basic Commands

1. What is SQL?

  • SQL stands for Structured Query Language.
  • SQL is used to store, retrieve, update and delete data in databases.
  • It works with database tables.
  • Common SQL commands include CREATE, INSERT, SELECT, UPDATE, ALTER and DELETE.

Question No. 1

To create a table named students with the following fields and insert the following records.

Roll Number Name Address Grade
1 Dinesh Kathmandu 11
2 Suresh Pokhara 12
3 Sumina Kathmandu 12
4 Himal Dharan 11
5 Sangita Palpa 11

Tasks

  1. Add the field named DOB in the above table.
  2. Insert five records in the students table.
  3. Display Name and Address of the students from the students table.
  4. Display all the records of students whose Address = "Kathmandu".
  5. Update any one record of the students table.
  6. Delete the record of the student named "Suresh".

2. Create the Students Table

The following SQL statement creates a table named students.

CREATE TABLE students (
    Roll_Number INT PRIMARY KEY,
    Name VARCHAR(30),
    Address VARCHAR(30),
    Grade INT
);
Output:

Table students created successfully.

3. Insert Records

Insert the given student records into the table.

INSERT INTO students
VALUES
(1, 'Dinesh', 'Kathmandu', 11),
(2, 'Suresh', 'Pokhara', 12),
(3, 'Sumina', 'Kathmandu', 12),
(4, 'Himal', 'Dharan', 11),
(5, 'Sangita', 'Palpa', 11);
Output:
Roll_Number Name Address Grade
1 Dinesh Kathmandu 11
2 Suresh Pokhara 12
3 Sumina Kathmandu 12
4 Himal Dharan 11
5 Sangita Palpa 11

4. Add a New Field (DOB)

Add a new column named DOB to the students table.

ALTER TABLE students
ADD DOB DATE;
Output:
Roll_Number Name Address Grade DOB
1 Dinesh Kathmandu 11 NULL

5. Display Name and Address

SELECT Name, Address
FROM students;
Output:
Name Address
Dinesh Kathmandu
Suresh Pokhara
Sumina Kathmandu
Himal Dharan
Sangita Palpa

6. Display Students from Kathmandu

SELECT *
FROM students
WHERE Address = 'Kathmandu';
Output:
Roll_Number Name Address Grade
1 Dinesh Kathmandu 11
3 Sumina Kathmandu 12

7. Update a Record

Change Dinesh's address from Kathmandu to Lalitpur.

UPDATE students
SET Address = 'Lalitpur'
WHERE Roll_Number = 1;
Output:
Roll_Number Name Address Grade
1 Dinesh Lalitpur 11

8. Delete a Record

Delete the student whose name is Suresh.

DELETE FROM students
WHERE Name = 'Suresh';
Output:
Roll_Number Name Address Grade
1 Dinesh Lalitpur 11
3 Sumina Kathmandu 12
4 Himal Dharan 11
5 Sangita Palpa 11

9. Insert DOB Values

Update the DOB column for all students.

UPDATE students
SET DOB = '2008-01-15'
WHERE Roll_Number = 1;

UPDATE students
SET DOB = '2007-05-20'
WHERE Roll_Number = 2;

UPDATE students
SET DOB = '2007-09-10'
WHERE Roll_Number = 3;

UPDATE students
SET DOB = '2008-03-25'
WHERE Roll_Number = 4;

UPDATE students
SET DOB = '2008-11-18'
WHERE Roll_Number = 5;
