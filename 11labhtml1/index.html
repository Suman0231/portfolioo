HTML Beginners Tutorial

1. Structure of an HTML Page

This is the basic skeleton of any HTML webpage:

 CODE:
<!DOCTYPE html>
<html>
<title>YOUR NAME HERE</title>
<header>My First Webpage</header>

<body>
WE ARE LEARNING HTML
</body>

<footer>copyright @ YOURNAME</footer>
</html>

2. Paragraph & Heading Tags

Paragraphs:

 CODE:
<p>My first paragraph.</p>
<p>My second paragraph.</p>

Headings:

 CODE:
<h1>My First Heading</h1>
<h2>My Second Heading</h2>
<h3>My Third Heading</h3>
<h4>My Fourth Heading</h4>
<h5>My Fifth Heading</h5>
<h6>My Sixth Heading</h6>

3. Line Break & Horizontal Line

 CODE:
<p>My third<br>paragraph.</p>
<hr>

4. Add Background Color

 CODE:
<body style="background-color:blue;">

5. Add Background Image

 CODE:
body {
  background-image: url('img_girl.jpg');
}

6. Add an Image

 CODE:
<img src="imagename.jpg" 
     alt="HTML Image description" 
     width="128" 
     height="128">

8. Formatting Text

 CODE:
<p><b>This text is bold</b></p>
<p><i>This text is italic</i></p>
<p>This is<sub> subscript</sub> 
and <sup>superscript</sup></p>

<strong>Important text</strong>
<em>Emphasized text</em>
<mark>Marked text</mark>
<small>Smaller text</small>
<del>Deleted text</del>
<ins>Inserted text</ins>

<blockquote>This is a blockquote example.</blockquote>

<pre>
This is preformatted text.
Spaces and line breaks are kept exactly.
</pre>

9. Ordered & Unordered Lists

 CODE:
<ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>

<ol>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

10. List Style Types

 CODE:
<ul style="list-style-type:circle;">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>

<ol type="a">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

Also try other different types of ordered and unordered lists.

  • Unordered List Types:
    • Disc
    • Circle
    • Square
    • None
  • Ordered List Types:
    • type="1"
    • type="A"
    • type="I"
    • type="i"

11. Links in HTML

 CODE:
<a href="https://www.youtube.com/" target="_blank">Visit Youtube!</a>

Also try self, parent targets.

12. Sending Mail/ Calling in HTML

 CODE:
<a href="mailto:yourmailId">Click to send mail.</a>
<a href="telto:yourPhoneNumber">Click to Call.</a>

12. iframes in HTML

 CODE:
<iframe src="file.html" height="200" width="300" title="Iframe Example"></iframe>
