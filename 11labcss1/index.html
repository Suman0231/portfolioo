CSS Tutorial – Basics & Selectors

1. What is CSS?

  • CSS stands for Cascading Style Sheets
  • CSS is used to style HTML elements
  • It controls colors, layout, fonts
  • External styles are saved in .css files

2. CSS Syntax

A CSS rule consists of a selector and a declaration 
selector {
  property: value;
}

3. Full CSS Example

<!DOCTYPE html>
<html>
<head>
<style>

body {
  background-color: lightblue;
}

h1 {
  color: white;
  text-align: center;
}

p {
  font-family: verdana;
  font-size: 20px;
}

</style>
</head>

<body>

<h1>My First CSS Example</h1>
<p>This is a paragraph.</p>

</body>
</html>
Output:

My First CSS Example

This is a paragraph.

4. Element Selector

<!DOCTYPE html>
<html>
<head>
<style>

p {
  text-align: center;
  color: red;
}

</style>
</head>

<body>

<p>Every paragraph will be affected</p>
<p>Me too!</p>

</body>
</html>
Output:

Every paragraph will be affected

Me too!

5. ID Selector

<!DOCTYPE html>
<html>
<head>
<style>

#para1 {
  text-align: center;
  color: red;
}

</style>
</head>

<body>

<p id="para1">Hello World!</p>
<p>This paragraph is not affected</p>

</body>
</html>
Output:

Hello World!

This paragraph is not affected

6. Class Selector

<!DOCTYPE html>
<html>
<head>
<style>

.center {
  text-align: center;
  color: red;
}

</style>
</head>

<body>

<h1 class="center">Heading</h1>
<p class="center">Paragraph</p>

</body>
</html>
Output:

Heading

Paragraph

Try this: Replace .center with p.center 👉 Only paragraphs will change, not the heading.

7. Universal Selector

<!DOCTYPE html>
<html>
<head>
<style>

* {
  text-align: center;
  color: blue;
}

</style>
</head>

<body>

<h1>Hello World</h1>
<p>This is a paragraph</p>

</body>
</html>
Output:

Hello World

This is a paragraph
