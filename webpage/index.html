Paradise Resort Website Tutorial

1. Basic HTML Structure

This is the starting structure of the web app:

CODE:

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <title>Paradise Resort</title>
</head>

<body>

</body>

</html>
This creates the main structure of the webpage.

2. Add a Navigation Bar

Create a menu for navigating different sections.

CODE:

<nav>
  <a href="#home">Home</a>
  <a href="#rooms">Rooms</a>
  <a href="#contact">Contact</a>
  <a href="#location">Location</a>
</nav>
The navbar helps users move between sections smoothly.

3. Style the Navbar with CSS

CODE:

nav {
  background-color: darkgreen;
  padding: 15px;
  text-align: center;
  position: sticky;
  top: 0;
}

nav a {
  color: white;
  text-decoration: none;
  margin: 15px;
  font-size: 18px;
}

nav a:hover {
  color: yellow;
}
The navbar becomes sticky and changes color when hovered.

4. Create the Home Section

CODE:

<section id="home">

<h1>Welcome to Paradise Resort</h1>

<p>
Enjoy beautiful views, comfortable rooms,
and relaxing vacations at our resort.
</p>

</section>
This is the main introduction section of the website.

5. Add an Image

CODE:

<img src="https://images.unsplash.com/photo-1506744038136-46273834b3fb"
     alt="Resort Image">

6. Style Sections

CODE:

section {
  background-color: white;
  margin: 20px;
  padding: 40px;
  border-radius: 10px;
}
Each section now appears like a clean card layout.

7. Create a Rooms Section

CODE:

<section id="rooms">

<h2>Our Rooms</h2>

<ul>
  <li>Deluxe Room</li>
  <li>Family Suite</li>
  <li>Sea View Room</li>
</ul>

</section>
Lists are useful for displaying room types or services.

8. Add Contact Information

CODE:

<a href="mailto:paradiseresort@example.com">
Send Email
</a>

<a href="tel:+9779800000000">
Call Now
</a>
Users can directly email or call from the webpage.

9. Add External Website Link

CODE:

<a href="https://www.booking.com"
   target="_blank">

Book Now

</a>
The link opens another website in a new tab.

10. Add Google Map Using iframe

CODE:

<iframe
src="https://maps.app.goo.gl/dAFdf6N6eP2jm6g26">
</iframe>
iframe is used to embed maps, videos, or other webpages.

11. Style Images and iframe

CODE:

img {
  width: 100%;
  border-radius: 10px;
}

iframe {
  width: 100%;
  height: 300px;
  border: 2px solid gray;
  border-radius: 10px;
}
Images and maps become responsive and attractive.

12. Add Footer

CODE:

<footer>
<p>

</p>
</footer>
