2. PHP Case Sensitivity

In PHP, keywords (such as if, else, while, echo), classes, and functions are not case-sensitive.

The following example shows that all echo statements work the same way: 

<html>
<body>

?>

</body>
</html>

4. Rules for PHP Variables

PHP variables are used to store data. The rules for declaring variables are:

  • A variable starts with the $ sign
  • Variable name must start with a letter or underscore
  • Variable name cannot start with a number
  • Only alpha-numeric characters and underscores are allowed
  • Variable names are case-sensitive

5. PHP Variables: String, Integer, and Float

PHP supports different data types such as string, integer, and float. 

<?php

echo "string is: $str <br/>";
echo "integer is: $x <br/>";
echo "float is: $y <br/>";
?>

7. PHP Program to Find Factorial of a Number

This program accepts a number from the user and calculates its factorial using a for loop. 

<html>
<head>
<title>Factorial Program</title>
</head>
<body>

<b>Factorial Calculation</b><br/><br/>

<form method="POST">
Input a Number:
<input type="number" name="number"><br/>
<input type="submit" value="Calculate">
</form>

<?php
if($_SERVER["REQUEST_METHOD"]=="POST")
{
    $n = $_POST['number'];
    $fact = 1;

    for($i=1; $i<=$n; $i++)
    {
        $fact = $fact * $i;
    }
    echo "The factorial of $n = $fact";
}
?>

</body>
</html>

10. PHP Program to Calculate Simple Interest

This program calculates the simple interest based on principal, time, and rate entered by the user. 

<html>
<head>
<title>Calculate Simple Interest</title>
</head>
<body>

<b>Calculate Simple Interest</b><br/><br/>

<form method="POST">
Enter Principal:
<input type="number" name="p"><br/>

Enter Time:
<input type="number" name="t"><br/>

Enter Rate:
<input type="number" name="r"><br/>

<input type="submit" value="Calculate SI">
</form>

<?php
if($_SERVER["REQUEST_METHOD"]=="POST")
{
    $p = $_POST['p'];
    $t = $_POST['t'];
    $r = $_POST['r'];

    $si = ($p * $t * $r) / 100;
    echo "Simple Interest = $si";
}
?>

</body>
</html>
